## Технологии

*   React
*   TypeScript
*   dnd-kit
*   mantine/core
*   CSS Modules

## Как запустить (примерно)

1.  **Установите зависимости:**
    ```bash
    npm install
    # или
    yarn install
    ```
2.  **Запустите проект:**
    ```bash
    npm start
    # или
    yarn start
    ```

Компонент `RuleBuilder` можно будет импортировать и использовать в вашем React-приложении.
